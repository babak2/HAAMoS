Class SortComparatorUtils

public class SortComparatorUtils

Utility class consisting of comparator implementations of different object/class which compare two arguments for order (to be used by Collection.sort method). For an ascending order, for example, the compare method of the Comparator interface should return a negative integer, zero, or a positive integer as the first argument is less than, equal to, or greater than the second.
