src.nz.ac.auckland.compass.haamos.space
Class AreaUnit

public class AreaUnit
implements GisAgent, OpenMapAgent

This class represents one possible form of spatial unit that corresponds to the SNZ Area Unit and contains census-based population.

In this model, the exact locations of households within area units (AU) are not specified (matter of concern).
